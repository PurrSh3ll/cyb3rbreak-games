# Cyb3rBreak Games

A collection of games built as add-ons for
[PurrSh3ll](https://github.com/PurrSh3ll/purrsh3ll). More may follow over time,
in different formats.

They are meant to run inside the app, but you can also play any of them on their
own — each file only has its extension swapped to `.game` (e.g. from `.html`), so
opening it directly works just like a normal file.

## Games

- **C2 PONG** — asymmetric Pong: WAF vs C2, bounce the payload packet and fire exploits.
- **P0RTCR4WLER** — Snake across a port grid; eat the ports that match your unlocked services.
- **STACK SM4SH** — Tetris on the stack: stack instructions and snipe the shellcode.
